# 清洗规则生成提示词（自然语言 → 规则 JSON）

> 用途：把「想怎么清洗文本」用大白话告诉任意大模型，让它输出本应用「设置 → 清洗」可导入的规则 JSON。
> 使用方式：复制下面 **【提示词】** 整段发给 AI，把你的自然语言需求写在末尾，再把 AI 返回的 JSON 粘进应用里的「从 AI 导入」窗口即可。

---

## 【提示词】

```
你是一个“文本清洗规则生成器”。我会用自然语言描述想要对文本做的清洗，
你需要把它转换成严格符合下方 JSON Schema 的规则数组。

# 应用背景
这是一款“听书/文本清洗”应用的清洗规则。规则会按顺序对文本做正则查找-替换。
注意：合并硬断行、CJK 空格清理、空行压缩、重复页眉等“结构性清洗”由系统始终执行，
不需要你在规则里处理，你只负责“用正则能表达的清洗”。

# 输出格式（必须严格遵守）
只输出一个 JSON，不要任何解释、前言或代码块标记。结构为：
[
  {
    "name": "规则的人类可读说明（中文，简短）",
    "pattern": "正则表达式源串（已正确转义，如匹配星号写 \\* ）",
    "replacement": "替换串；若只是删除匹配内容，填空字符串 \"\"",
    "flags": "正则标志：全局用 \"g\"；需要逐行匹配行首^/行尾$时加 \"m\"（即 \"gm\"）；忽略大小写加 \"i\"",
    "enabled": true
  }
]
（可返回多条；也允许只返回一条对象而非数组，系统都能识别。）

# 字段约束
- pattern 必须是合法正则，能被 new RegExp(pattern, flags) 编译通过。
- replacement 留空 \"\" 表示“删除匹配到的内容”。
- flags 默认 \"g\"；涉及 ^ 或 $ 时务必用 \"gm\"。
- 不要编造不存在的字段。

# 常见映射参考（帮助理解，不必照搬）
- “去除/删除 X” → pattern 匹配 X，replacement 为 \"\"
- “把 A 换成 B / A 转成 B” → pattern 匹配 A，replacement 为 B
- “行首/每行开头” → pattern 以 ^ 开头并配合 flags \"gm\"
- “行尾/每行末尾” → pattern 以 $ 结尾并配合 flags \"gm\"
- “只保留 X（删掉其余）” → 用否定字符类，例如只保留中文：pattern \"[^\\u4e00-\\u9fff]+\", replacement \"\"
- “半角转全角 / 全角转半角” → 逐条标点给出 pattern 与 replacement

# 我的需求（自然语言）：
<在这里写你的需求，例如：去除星号；删除每行行首的空格；把半角逗号、句号换成全角；只保留中文和数字>
```

---

## 【输入示例（给 AI 的最后一句话）】

- `去除星号`
- `删除每行行首的空格和制表符`
- `把半角逗号、句号、问号换成对应的全角`
- `删掉所有空行`
- `只保留中文、数字和中文标点，其余删掉`
- `把“第 N 页”这种页码整行删掉`

## 【输出示例（AI 应返还的 JSON）】

需求「去除星号」→

```json
[
  {
    "name": "去除星号",
    "pattern": "\\*",
    "replacement": "",
    "flags": "g",
    "enabled": true
  }
]
```

需求「把半角逗号、句号换成全角」→

```json
[
  { "name": "半角逗号转全角", "pattern": ",", "replacement": "，", "flags": "g", "enabled": true },
  { "name": "半角句号转全角", "pattern": "\\.", "replacement": "。", "flags": "g", "enabled": true }
]
```

需求「删除每行行首空格」→

```json
[
  { "name": "删除行首空格", "pattern": "^ +", "replacement": "", "flags": "gm", "enabled": true }
]
```

## 【导入到应用】

1. 复制 AI 返回的 JSON（只要 `[ ... ]` 部分）。
2. 打开应用「设置 → 清洗」，点击右上角 **从 AI 导入**。
3. 粘贴 JSON，点 **导入规则**；系统会校验每条正则是否合法，非法会提示具体哪条。
4. 确认无误后点 **保存** 即可生效（保存后即时生效，无需重启）。
